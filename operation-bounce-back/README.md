## ğŸ” Technology Stack

### Blockchain
- **Network**: Polygon (for low gas fees and scalability)
- **Standards**: ERC-721 (NFTs), ERC-20 (DE Tokens)
- **Framework**: Hardhat for development and testing
- **Libraries**: OpenZeppelin v5.0.1 for security

### Smart Contract Features
- **Access Control**: Role-based permissions for minting and distribution
- **Pausable**: Emergency stop mechanism
- **Royalty Standards**: ERC-2981 for perpetual royalty distribution
- **Metadata**: IPFS storage for decentralized NFT data
- **QR Code Integration**: Dynamic metadata linking to donation records

### Development Tools
- **Testing**: Hardhat with Chai assertions
- **Deployment**: Automated scripts for Mumbai testnet and Polygon mainnet
- **Verification**: Etherscan/Polygonscan contract verification
- **CI/CD**: GitHub Actions for automated testing

## ğŸ“… Quick Start Guide

### For Developers
1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables (see `.env.example`)
