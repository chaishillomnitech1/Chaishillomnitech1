# Monitoring Commands for Engagement Metrics

## Overview
This document outlines the commands and frameworks for monitoring engagement metrics effectively within the ScrollVerse community.

## Commands
1. **Active Users Command:**  
   - `GET /api/users?active=true`
   - This command retrieves the list of active users engaged in the platform.

2. **Engagement Rate Calculation:**  
   - Formula: 
     
     \[\text{Engagement Rate} = \frac{\text{Total Interactions}}{\text{Total Users}} \times 100\]\
   
3. **Feedback Loop Command:** 
   - `POST /api/feedback`
   - Sends feedback from users and logs it for analysis.
